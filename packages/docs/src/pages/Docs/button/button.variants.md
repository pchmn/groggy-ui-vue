## Button variants

There are 3 variants: `standard` (default), `outlined` and `flat`.

<!-- demo -->
<template>
  <div v-tw="'flex space-x-4'">
    <g-button>Standard</g-button>
    <g-button variant="outlined">Outlined</g-button>
    <g-button variant="flat">Flat</g-button>
  </div>
</template>
<!-- /demo -->
